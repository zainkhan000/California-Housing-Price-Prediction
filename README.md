# 🏠 California Housing Price Prediction

## 📌 Project Description

This project demonstrates a complete **Machine Learning regression workflow** designed to estimate median housing prices in California. The model is built using the **California Housing dataset available in Scikit-learn**.

The project covers the entire ML pipeline including:

* Loading and preparing the dataset
* Performing **Exploratory Data Analysis (EDA)**
* Studying relationships between features using **correlation analysis**
* Training a **Linear Regression model**
* Evaluating model performance with multiple regression metrics
* Visualizing prediction performance
* Saving the trained model using **Pickle** for later use

---

## 🛠 Technologies Used

The project is developed using the following tools and libraries:

* **Python**
* **Pandas**
* **NumPy**
* **Matplotlib**
* **Seaborn**
* **Scikit-learn**

---

## 📊 Model Evaluation Metrics

To assess the effectiveness of the regression model, the following metrics are used:

* **MAE (Mean Absolute Error)** – Measures the average prediction error
* **RMSE (Root Mean Squared Error)** – Indicates the standard deviation of prediction errors
* **R² Score (Coefficient of Determination)** – Shows how well the model explains variance in the data

---

## 📈 Data Visualizations

Several visualizations are included to better understand the dataset and model performance:

* **Distribution Plot of Target Variable** – Shows how housing prices are spread
* **Correlation Heatmap** – Displays relationships between features
* **Actual vs Predicted Values Plot** – Compares model predictions with real values
* **Residual Plot** – Helps analyze prediction errors
